## Laws of Set theory

Here are the laws of set theory shown in the image along with their names:

1. **Law of Double Complement**
   - $$\overline{\overline{A}} = A$$

2. **De Morgan's Laws**
   - $$\overline{A \cup B} = \overline{A} \cap \overline{B}$$
   - $$\overline{A \cap B} = \overline{A} \cup \overline{B}$$

3. **Commutative Laws**
   - $$A \cup B = B \cup A$$
   - $$A \cap B = B \cap A$$

4. **Associative Laws**
   - $$A \cup (B \cup C) = (A \cup B) \cup C$$
   - $$A \cap (B \cap C) = (A \cap B) \cap C$$

5. **Distributive Laws**
   - $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$
   - $$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$

6. **Idempotent Laws**
   - $$A \cup A = A$$
   - $$A \cap A = A$$

7. **Identity Laws**
   - $$A \cup \emptyset = A$$
   - $$A \cap U = A$$

8. **Inverse Laws**
   - $$A \cup \overline{A} = U$$
   - $$A \cap \overline{A} = \emptyset$$

9. **Domination Laws**
   - $$A \cup U = U$$
   - $$A \cap \emptyset = \emptyset$$

10. **Absorption Laws**
    - $$A \cup (A \cap B) = A$$
    - $$A \cap (A \cup B) = A$$

These laws form the foundation of set theory and are used extensively in various fields, especially in computer science for logic design, database theory, and algorithms.
